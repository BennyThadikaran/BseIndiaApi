=====
Usage
=====

Installation
------------

To use ``bse``, first install it using pip:

.. code:: console

   $ pip install bse

Example
-------

.. code-block:: python

   from bse import BSE

   bse = BSE()

   bse.exit() # close requests session

.. code-block:: python
   :caption: Using 'with' statement

   with BSE() as bse:
      pass
